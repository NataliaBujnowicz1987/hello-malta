### Web Application Development (Project 2019) - ‘Hello Malta’ - share your experience 

#### Used technologies: HTML5, CSS3, JS+jQuery, Node.js+Express.js, MySQL, JSON, XML. This application was built on Cloud9 application platform. 

##### INDEX PAGE 
###### jQuery Effects: show/hide header (by clicking the maltese cross); responsive menu; user can change the color of individual boxes; the one part of txt in the main header is changing every 1second (Hello/Enjoy/Visit/Love Malta); responsive slider with amazing photos.
###### Search bar connected with SQL database.
